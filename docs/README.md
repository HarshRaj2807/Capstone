# Project Documents

This folder keeps both source markdown and generated PDFs for easy reading/sharing.

## Source Files

- `*.md` files in this folder are the editable source documents.

## Generated Output

- `pdfs/` contains generated PDF versions of the markdown documents.
- `html/` contains intermediate HTML files used to build PDFs.

## Additional Organization

- `reports/` for project report files
- `diagrams/` for ER diagrams and architecture/design diagrams
- `other/` for any additional important documents
